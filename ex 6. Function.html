    새로운 함수를 만들 필요없이, 기존의 함수로 여러 가지의 활용을 만들 수 있는 방법은 무엇인가?
    -> Return

    function practice(a,b){
    return a*b
    }
    var x = practice(3,4);
    var y = practice(5,6);
    document.write('< div style = "font-size:2rem">'+(x*y)+'');
    document.write('< div style = "color:red">'+practice(12,12)+'');

    : 즉, 계산하는 기능만을 practice라는 함수가 구현함으로써, 원자화된 기능을 다양한 맥락에서 활용할 수 있게 된다.

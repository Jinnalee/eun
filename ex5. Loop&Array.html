var 동료 = ["일","이","삼"]
var i = 0;
while (i< 동료.length) { document.write('태그'+동료[i]+'/태그');
i = i + 1 }

< Consol에도 사용 가능 >

1. document.querySelector('a')
: querySelector라는 method/함수/명령은 'a'태그에 해당되는 첫번째 것만 가져옴
그러므로 'a'태그 전체는, (how to get multiple elements by CSS selector)

2. document.querySelectorAll('.class1.class2')
: console에 해당 함수를 입력하면, [a,a,a .. ]의 꼴, 배열과 같은 형태(정확히는 Nodelist)가 나옴.
즉, a 들의 리스트가 나온다. 옷(a)들이 들어있는 서랍(a들의 리스트)

    이걸 alist라는 변수로 두고, var alist = document.querySelectorAll('a');
  • console.log(alist[0])
    -> 괄호안의 결과가 콘솔 창에 출력되면서, 첫번째 링크가 뜬다.
  • alist.length : a(태그)의 개수

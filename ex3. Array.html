Array 배열

Syntax 문법

* 문자는 ""이지만, 배열은 [원소와 원소(element) 사이는 ,]
var 동료 =["가나다"] : '동료'라는 변수에 "가나다" 배열이라는 새로운 datatype이 담긴 것.
즉, "가나다"배열이라는 datatype에 '동료'라는 이름을 붙인것.
ex. var 동료 = ["Jina","Havana", "Won","Yul","Yohan"]

Get 배열 꺼내기

document.write(동료[0]) : 첫번째 자리값은 0번째(index0번)

->

document.write(동료[1]) : index 1

->

Array Add : 배열 내 Data 추가

동료.push("durian")

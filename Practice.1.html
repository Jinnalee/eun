HTML과 JavaScript

: 둘 다 컴퓨터 언어
1. HTML : 컴퓨터 프로그랭 언어는 아니다.
2. JavaScript : 컴퓨터 프로그래밍 언어이기도 함.
-> 프로그래밍이란?
프로그램에는 "순서"라는 의미가 내포됨. 순서를 만드는 행위가 "프로그래밍"
순서대로 여러 기능을 작동시키는 것이 컴퓨터를 사용하는 일반적 방법. 시간의 순서에 따라 실행되어야 할 기능을 프로그래밍 문법 언어에 따라 글로 적어두는 방식이 프로그램.
HTML은 웹페이지를 묘사하는 목적의 언어이기에, 시간의 순서에 따라 실행할 필요가 없음. 그러나 JS는 사용자와 상호작용하기 위한 언어이고, 그러므로 시간의 순서에 따라 웹브라우저의 여러 기능이 실행되어야하기 떄문에 프로그래밍의 형태를 띤다.
